A webapp to render chemical structures using the CDK.

Currently supported file types: jpeg, png, bmp, wbmp, gif, emf, pdf, svg, swf, ps, mol, smi, sdf

Source code available from http://code.google.com/p/cdk-export/

URL parameters:
NameDescription
structurea molecule string, such as smiles, mol etc.
highlighta molecule string, such as smiles, mol etc. Will be overlayed on the structure.
ida PubChem CID or SID.
idtypespecifies whether id is a "cid" or "sid".
filenamea filename for the export (forces browser download of the file).
exportan export format string. Parameters are name:value pairs separated by a comma. For a list of supported parameters click here.
